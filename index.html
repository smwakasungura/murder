Murder!

A GitHub game

Scenario: a murderer is on the loose! If your name turns red , you have been poisoned !

To get the antidote , you must use GitHub to change the code, and pick the next victim.

Instructions
  • All team members clone the github repo. Then begin the game. Cheatsheet
  • 1) Victim: create & checkout a branch in your name
  • 2) Open the index.html file
  • 3) Add the class 'antidote' to the h5 tag of your name
  • 4) Add the class 'poison' to the h5 tag of your victim's name
  • 5) Git add, commit & push your changes
  • 6) Open a pull request on GitHub
  • 7) Person2 review pull request, merge branch to main, and delete unneeded named branch
  • 8) In VSCode terminal/command line, victim to 'pull' on the main branch
  • 9) Repeat steps 1-9 so everyone in the group has a turn
